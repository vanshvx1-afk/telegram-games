Why Single File?
✅ No dependencies — Pure vanilla JS, no npm, no webpack
✅ Works anywhere — Just open in browser, instant play
✅ Easy to edit — All code in one place, Ctrl+F to find anything
✅ Fast loading — No multiple HTTP requests
✅ Offline ready — Service Worker caches the single file
✅ GitHub Pages friendly — Upload & go, no build step
